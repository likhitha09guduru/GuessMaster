# GuessMaster
This is my first GitHub Project.

GuessMaster - A Competitive Number Guessing Game:
In this multiplayer game, a Guesser picks a secret number while 3 players compete to predict it. An Umpire validates all guesses and crowns the accurate predictor as the GuessMaster for that round. 

Game Rules:
1) The Guesser selects a secret number between 1 and 100
2)Each of the 3 Players submits their guess
3)The Umpire compares all guesses to the secret number
4)If a player's guess matches the secret number, they are declared the GuessMaster of that round
5)If multiple players guess correctly, all are declared GuessMasters
6)If no one guesses correctly, no GuessMaster is declared for that round
